* Description :unfold:
This module provides backwards compatibility with v2 of Doom's core. It
reinstates old APIs, variables, packages, and configuration that are slated for
removal in v3. This exists to give you more time to adapt to the changes.

For a full table of what's been changed and what to use instead, see the Usage
section below.

** Module flags
- -better-jumper ::
  Don't install or configure ~better-jumper~.
- -general ::
  Don't install ~general~ and don't provide the ~map!~ macro.
- -projectile ::
  Don't install or configure ~projectile~.
- -smartparens ::
  Don't install or configure ~smartparens~.
- -use-package ::
  Don't install or configure ~use-package~, or define ~use-package!~ or
  ~use-package-hook!~.
